搜索语法 / Search Syntax
  • source:codex — 按来源筛选 / filter by source
  • type:document — 按类型筛选 / filter by type
  • date:2024-06-01 — 按日期筛选 / filter by date
  • from:2024-06-01 — 起始日期 / start date
  • to:2024-06-30 — 结束日期 / end date
  • title:REST — 按标题筛选 / filter by title
  • keyword:auth — 按关键词筛选 / filter by keyword
  • path:src — 按路径筛选 / filter by path
  • "exact phrase" — 精确短语匹配 / exact phrase match
  • -keyword — 排除包含该词的结果 / exclude term

多个条件用空格分隔，例如：source:codex type:document "REST API"
